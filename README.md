# BE THE HERO

Sistema de doações para instiuições de caridade, projeto do treinamento Oministack XI realizado pela Rocketseat.

## Stack do projeto

**Api**
 - nodejs  - 14.20.1
 - knexjs  - 0.30.13
 - sqlite3 - 4.1.4
 
**Web**
  - react - 16.13.1
  - axios - 0.20.0
